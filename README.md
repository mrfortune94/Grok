# Grok

A simple, elegant web application for exploring and understanding ideas.

## Features

- **Clean, Modern UI**: Responsive design that works on all devices
- **Interactive Input**: Enter text and get "grokked" insights
- **Lightweight**: Pure HTML, CSS, and JavaScript with no heavy dependencies

## Getting Started

### Prerequisites

- Node.js (for running the development server)

### Running the Application

1. Clone the repository:
   ```bash
   git clone https://github.com/mrfortune94/Grok.git
   cd Grok
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3000`

### Alternative: Open Directly

You can also open `index.html` directly in your web browser.

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/mrfortune94/Grok)

### Manual Deployment

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Deploy:
   ```bash
   vercel
   ```
